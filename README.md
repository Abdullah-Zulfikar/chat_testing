# Chat Testing App

This is a Next.js app scaffolded with TypeScript, Tailwind CSS, ESLint, and the App Router. It will display a list of contacts and a chat interface to send and receive messages.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Features

- Contact list display
- Chat interface for sending and receiving messages

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
